# Mapa do diretorio 08_tickets_ciss

## Objetivo

Guardar scripts vinculados a tickets CISS, principalmente quando a entrega altera mais de uma tabela, rotina ou regra.

## Conteudo

- `CISS-172088.sql`: script associado ao ticket CISS-172088.
- `CISS-174814.sql`: script associado ao ticket CISS-174814.
- `CISS-175870.sql`: script associado ao ticket CISS-175870.
- `MAPA_DIRETORIO.md`: este mapa local.

## Regras locais

- Nomeie scripts pelo numero do ticket sempre que esse for o identificador principal.
- Se um ticket virar uma solucao permanente de um assunto especifico, mova ou copie a versao final para o diretorio tematico correspondente.
- Evidencias relacionadas ao ticket devem manter o numero do ticket no nome.
